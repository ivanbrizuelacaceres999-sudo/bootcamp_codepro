# 🐍 Bootcamp CodePro — Fundamentos de Python

Colección de ejercicios y desafíos resueltos durante el bootcamp de programación **CodePro**, enfocado en los fundamentos de **Python**.

## 📚 Contenido
- **`day_1.ipynb`** — Variables, tipos de datos, entrada/salida y operadores
- **`dia_2.ipynb`** — Condicionales, bucles y funciones (challenges prácticos)
- **`calendario.md`** — Plan de estudios del bootcamp

## 🎯 Temas practicados
- Estructuras condicionales (`if` / `elif` / `else`)
- Validación de datos ingresados por el usuario
- Funciones y lógica de control
- Resolución de problemas tipo *challenge* con tiempo limitado

## 🛠️ Tecnologías
- **Python 3**
- **Jupyter Notebook**

## ▶️ Cómo abrir
```bash
git clone https://github.com/ivanbrizuelacaceres999-sudo/bootcamp_codepro.git
cd bootcamp_codepro
jupyter notebook
```
